VSC - Version Control System,
это программа, которая позволяет сохранять
промежуточные версии исходных кодов в контрольных точках,
и в любой момент возвращаться к этим контрольным точкам.
SVN
Subversion
Mercury
Git/GitHub
Git - система контроля версий
GitHub - облачное хранилище.

Repository - это хранилище исходных кодов.

Commit - это контрольная точка, 
в которую сохраняется текущее состояние репозитория.
Commit message - в нем описывают что именно было сделано в исходных кодах,
				 и каких результатов мы добились.
				 Commit message обязательно должен написать человек,
				 это никак нельзя автоматизировать.

fetch, pull и push
fetch - скачивает изменения с удаленного репозитория на локальный, 
		и НЕ распаковывает их;
pull  - скачивает и распаковывает изменения с удаленного репозитория в локальный;
push  - отправляет изменения с локального репозитория на удаленный;

Branch - это последовательность Commit-ов.
master - это главная ветка, в которой идет mainstream разработки.
merge  - слияние.
